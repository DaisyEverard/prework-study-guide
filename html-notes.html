HTML

Semantic HTML:

  • Semantic HTML Elements defined by use
  • header navigational links, introductory content, h1-h6. use h1 only once on page
  • nav For navigation links (menu, table of content...), On own or in header,
  • main Section where main content appears
  • footer Contact info, copyright, terms of use, site map, ref to top of page links.
  • section Chapters, heading, theme groupings eg. intro; news; contact info
  • article stand-alone content. articles, blogs, comments, magazines
  • aside Additional info. Bibliographies, endnotes, comments, pull quotes, editorial sidebars.
  • figure Encapsulates images and figcaptions.
  • figcaption Adds captions. Moves with figure unlike p elements
