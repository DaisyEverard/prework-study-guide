Formatting

from Google's standard guidelines

Indentation:

  • Two spaces per indentation level (child/parent)
  • Use spaces, not tabs
  • Make elements and attributes all lower case
  • Dont leave extra space at end of line

Line Length

  • Keep lines short as possible
  • Max line length 80ch unless:

    • all metadata tags must be on one line

    URLs can't have line breaks. put them on their own line within href attribute

    keep consistent with formatting of old file

    line break will change meaning of code


  • For images, always specify alt, width, and height
  • Dont put spaces around "="
  • Only add blank lines between large/logical blocks for readability
  • Use lower case file names
